# Data, AI & Operations Product Portfolio

Streamlit portfolio for Ritika Garg, focused on AI automation, data products, operations analytics, and leadership dashboards.

## Streamlit Cloud settings

```text
Repository: ritikagarg0903/data-ai-operations-portfolio
Branch: main
Main file path: app.py
```

## Cost

This app is designed to stay free to host and view. It uses Streamlit, Plotly, and static portfolio content. It does not call paid APIs at runtime.
